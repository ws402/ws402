## [0.1.0] - 2025-01-XX

### Added
- Initial release of WS402 library
- Core WS402 protocol implementation
- WebSocket session management
- Real-time usage tracking
- Automatic refund system
- Payment provider interface
- MockPaymentProvider for testing
- BasePaymentProvider template
- Express middleware helpers
- TypeScript support with full type definitions
- Comprehensive README documentation
- Working examples (server + client)
- Contributing guidelines
- MIT License

### Features
- Pay-as-you-go pricing model
- Upfront payment with automatic refunds
- BASE blockchain integration ready
- Session tracking (time, bytes, messages)
- Configurable pricing and limits
- Event system for monitoring
- User ID extraction from requests

### Examples
- Basic WebSocket server
- HTML/JS client demo
- Multiple payment provider templates

### Documentation
- Full API reference
- Quick start guide
- Integration examples
- Payment provider guide
- Contributing guide
